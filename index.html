Start Holding Giveaways!!

!ghelp - shows the available commands


!gstart <time> [prize] - starts a giveaway with the provided number of seconds. For example, !gstart 30 Steam Code would start a 30-second giveaway for a Steam Code! To use minutes instead of seconds, simply include an "m" in the time ~ !gstart 3m would be a 3-minute giveaway.


!gend - ends the most recent giveaway in the current channel. If you want to end an older/more sepcific giveaway, use !gend [messageId]


!greroll - picks a new winner from the latest giveaway. If you want to reroll an older giveaway, you can use !greroll [messageId]
